-->Privacy-Preserving Data Linkage Framework for Secure Agriculture Research 🌾

A secure data processing framework designed to protect sensitive agricultural data while enabling meaningful analytics for research and policy development.

This project implements privacy-preserving techniques using the automated selection of attributes and even rule based,Federated Learning, PCA, and Gaussian Mixture Models (GMM) along with a Flask-based interactive dashboard.

-->Project Overview

Agricultural datasets often contain sensitive farmer information, such as land area, crop production, income, and location.
Directly sharing such data can lead to privacy risks and misuse.

This project proposes a privacy-preserving framework that:

--Detects sensitive attributes automatically

--Protects sensitive information through transformation

--Performs analytics without exposing raw private data

--Enables researchers to analyze agricultural patterns safely

-->Key Features

--Secure privacy-preserving data processing

--Automatic sensitive attribute detection

--User-controlled attribute selection

--Federated Autoencoder for privacy-aware feature learning

--PCA for dimensionality reduction

--Gaussian Mixture Model clustering

--Privacy vs Utility analysis

--Interactive Flask web dashboard

--User authentication system (Login + Register)
